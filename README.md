# Personal URL Vault

Простое веб-приложение для хранения URL-адресов с заметками. Разработано как демонстрационное приложение для магистерской диссертации.

## Требования

- Java 21
- Maven 3.9+

## Запуск

```bash
mvn clean package
java -jar target/url-vault-*.jar
# или
mvn spring-boot:run
```

Приложение запускается на [http://localhost:8001/](http://localhost:8001/).

## Ключевые эндпоинты

### Web (Thymeleaf)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Главная страница со списком закладок и поиском |
| GET | `/add` | Форма добавления закладки |
| POST | `/add` | Сохранение новой закладки |
| POST | `/delete/{id}` | Удаление закладки |
| GET | `/go/{id}` | Редирект на сохранённый URL |

### REST API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/bookmarks` | Список всех закладок (JSON) |
| GET | `/api/bookmarks/{id}` | Одна закладка по id |
| POST | `/api/bookmarks` | Создать закладку |
| DELETE | `/api/bookmarks/{id}` | Удалить закладку |
| GET | `/api/health` | Статус приложения |

### Поиск

`GET /?q=текст&tag=тег` — фильтрация по подстроке в названии и/или точному тегу.

## Интеграция со шлюзом безопасности

Приложение разрабатывается для работы за прокси-шлюзом ML Security Gateway на порту 8080.
